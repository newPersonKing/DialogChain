# DialogChain

#### 介绍
 DialogChian 是根据责任链设计思想，用以优雅解决App 首页弹窗优先级编排问题的方案。代码比较简单，暂不支持远程依赖，建议直接ctrl+c+v!这样还能根据自己需求去魔改源码以适应各种水土不服。设计思想及说明详见文档：https://www.jianshu.com/p/42851f1dc050。
